Welcome to Alexandria

Alexandria is a collection of tools for Grasshopper for Rhino designed to streamline the creation of parametric geometries, arrays, and patterns.

Visit About Alexandria to learn more about Alexandria, or see the Installation Guide to get started.

About Alexandria

Alexandria is a collection of tools for Grasshopper for Rhino which allow users to input simple information such as a open/closed curve, a point, number slider, etc. and create complex 2D and 3D geometries, arrays, attractors, and patterns.

This is a living project, and it will continue to grow as we use it. The more we use it, the more it will grow. User feedback is encouraged, so please reach out with any questions, ideas, and suggestions.

Installation Guide

Follow these instructions to install the latest release of Alexandria on your own device. We are currently in beta, so some features may be limited, or you may encounter issues. Rest assured, we are working tirelessly to create a stable, functional public release of Alexandria as soon as possible.

  1. Open an existing Rhino file, or create a new one.
  2. Once you have a Rhino file open, type "Grasshopper" into your command bar. This will open the Grasshopper for Rhino Plugin. Open an existing GH file, or create a new one.
  3. In Grasshopper, go to File > Special Folders > User Object Folder.
  4. Open a new File Explorer tab, unzip the file you downloaded from our website.
  5. Copy all the contents from the newly extracted folder into your User Objects Folder.

As Alexandria is updated, users must repeat this process to install the most current version of the app.

Frequently Asked Questions

WHAT IS A GRASSHOPPER DEFINITION?

A Grasshopper definition is a collection of interconnected components that define a specific parametric process or design in Rhino. It consists of visual programming elements such as inputs (parameters), logic (components), and outputs (results), and allows users to create complex forms, structures, or behaviors by manipulating geometry and data.

WHAT IS A CLUSTER WITHIN GRASSHOPPER?

A Cluster in Grasshopper is a way to group a set of components together into a single, reusable unit. This allows you to simplify your definition, improve organization, and create modular systems.

WHAT IS A GRASSHOPPER PARAMETER? WHAT DO THEY DO?

Parameters in Grasshopper are the values that define the input or output of a component. They can be numbers, geometry, or other types of data, and they are crucial for manipulating and controlling the flow of information through your definition.

WHAT KINDS OF PARAMETERS CAN BE USED AS INPUT?

Common input parameters include number sliders, geometry, vectors, booleans, strings, lists, and colours.

WHAT KINDS OF PARAMETERS WILL BE OUTPUT?

Output parameters can include geometry, vectors, numbers, lists, booleans, and text (strings), which define the results of a Grasshopper definition.

HOW DO I MOVE A GRASSHOPPER DEFINITION INTO RHINO?

To move a Grasshopper definition into Rhino, you can "bake" the geometry created in Grasshopper into Rhino's modeling environment. Right-click the output geometry and select "Bake" from the context menu.
